# 變更紀錄(Changelog)

本專案為原外掛「Hooper AI 2.0」(Bilibili UP 主「小壕h」,`com.mediastorm.subfix`)的繁體中文(台灣)在地化 + 改名版「路邊野貓 AI(AlleyCat)」。版號沿用原作的主版本,後綴 `-tw` 表示繁體台灣版。

## v2.3-tw

- 翻譯選單第 3 項旗標由 🇨🇳 改為 🇹🇼。
- 糾錯/翻譯提示詞新增指示:**一律輸出繁體中文(台灣用語、台灣標點符號)**,不得轉簡體。
- 「的/地/得」改為**台灣寬鬆版**:副詞用「的」(慢慢的、好好的)一律保留,不再強推「地」;只修正明顯的「得」補語誤用。
- 任務選項「的地得專項檢測」更名為「**的得專項檢測**」。
- 口頭禪清理新增**行首發語詞「那」**的安全規則(僅當「那,」「那 」這種獨立發語詞才移除,「那個/那裡」等指示詞不動)。
- 提示詞用語在地化:「地道」→「道地」。

## v2.2-tw

- 新增**口頭禪一鍵清理**按鈕(精修工具面板):移除嗯/呃/那個那個等語助詞與口吃疊詞,出修改報告、可撤回。

## v2.1-tw(在地化基底)

- 以 OpenCC `s2twp` 全檔簡轉繁 + 台灣慣用詞,並人工校對影視/技術用語。
- 品牌改名:`Hooper AI 2.0` → `路邊野貓 AI`(代號 AlleyCat);桌面備份夾、匯出檔名前綴、設定檔名同步改為 AlleyCat / `.alleycat_config`。檔名/選單名仍為 `SubFix`。
- 修正:原誤標的 Lua 版本(腳本使用 `goto`,屬 Lua 5.2+;以 Lua 5.4 `luac -p` 通過整檔語法檢查)。
- 提供以 bomutils 重新打包、結構驗證過的 macOS `.pkg`。

> 著作權仍屬原作者;本版僅作在地化與功能調整,公開散布前請取得原作者同意並標註出處。
